📖 デジログ 使い方ガイド

🚛 基本の流れ
① ドライバー選択（画面中段）
② 出庫ボタン → オドメーター入力
③ 荷積・荷降・運転開始を繰り返す
④ 帰庫ボタン → オドメーター入力
⑤ 日報出力で送信
▼ 操作パネル（メイン）
出庫 … 1日の運行開始。オドメーター入力あり
📋 点呼 … アルコール・健康・車両チェックを記録
📦 荷積 … 荷積地を選択/GPS自動取得
📥 荷降 … 荷降地を選択。実車→空車の確認あり
休憩 … 休憩開始を記録
運転開始 … 運転再開を記録。荷積後は自動で実車に
⏸ 待機 … 荷待ちなどの待機時間を記録
🛣 高速 … IC名と料金を記録
⛽ 給油 … リッター数と金額を記録
💧 洗車 … 洗車を記録
🔧 点検 … 点検を記録
帰庫 … 1日の運行終了。オドメーター入力あり
🔄 道路種別・積載状態
道路種別 … 一般道 / 高速道 を切替
積載状態 … 空車 / 実車 を切替
※ 荷積→運転開始で自動的に実車に切替
※ 荷降後の確認で空車に切替可能
📍 GPS機能
自動地名取得 … 荷積/荷降時にGPSで住所・施設名を自動入力
位置情報共有 … 管理者モードで各ドライバーの位置を確認可能
※ 事故やトラブル発生時の迅速な対応に活用
📤 日報出力・共有
通常版 … 全ログ・給油/高速明細付きの詳細日報
簡易版 … LINE送信向きのコンパクト日報
走行距離 … オドメーター未入力時は手動入力も可能
送信方法 … 共有シート / メール / CSV / テキスト / コピー
📊 分析・集計
📊 集計 … 週間・月間の走行距離/給油/高速料金
⛽ 燃費評価 … 燃費をスコアリングして評価
📅 履歴 … 過去の運行ログを日付別に閲覧
👁 管理者モード
全ドライバーの状況をリアルタイムで一覧表示
表示内容 … ステータス・積載・道路・走行距離・最終操作
📍 位置確認 … Google Mapsで位置を表示
🔄 最新に更新 … 手動リフレッシュ
⚙ 設定
👤 ドライバー管理 … ドライバーの追加・編集・削除
🚛 車両台帳 … 車両情報の登録・車検期限管理
💾 バックアップ/復元 … 全データのJSON保存と復元
📋 運行予定 … 本日の運行予定を登録
⚠ 安全アラート
連続運転 … 4時間超で警告
拘束時間 … 13時間超で警告
インターバル … 前回帰庫から9時間未満で警告
実車帰庫 … 実車のまま帰庫で注意喚起
📋 ログ
本日の全操作ログを時系列で表示
各ログをタップすると編集・削除が可能
